# Project Overview
This repository contains the software development for an autonomous mobile robot navigation system. Developed as part of the Robotics and Software Engineering curriculum, the project focuses on enabling a robotic agent to map environments, avoid obstacles, and autonomously navigate from a starting point to a designated target.

###  Tech Stack
* **Language:** Python
* **Key Fields:** Mobile Robotics, Path Planning, Obstacle Avoidance, Sensor Integration

###  Core Capabilities Implemented
* **Path Planning & Navigation:** Algorithms designed for efficient route calculation in dynamic environments.
* **Obstacle Avoidance:** Real-time processing of sensor data to detect and steer away from unexpected barriers.
* **Localization:** Utilizing sensory input to keep track of the robot's current coordinates within the simulation space.
